# 帖子精细化管理「表格/卡片视图切换」增量文档（2026-04-13）

## 1. 变更目标

页面：`/admin/posts`

在不影响现有管理能力的前提下，新增“表格 / 卡片”视图切换，提升不同使用习惯下的可读性与操作效率。

## 2. 前端已完成

1. 新增视图切换控件：`表格`（默认）/`卡片`
2. 卡片视图展示字段：
- `id`
- `title`
- `summary/content` 摘要
- `author`
- `boardName`
- `format`
- `visibility`
- `status`
- `tags`
- `updatedAt`
3. 卡片视图保留完整操作按钮：
- 编辑
- 置顶/取消置顶
- 加精/取消加精
- 发布/下架
4. 原有筛选、分页、每页条数在两种视图中共用同一数据源和状态。

## 3. 后端影响

本次无新增后端接口。

仍使用：
- `GET /api/v1/posts`（分页+筛选）
- `PATCH /api/v1/posts/{postId}`（编辑与状态策略操作）

## 4. 联调验收项

1. 在 `/admin/posts` 切换“表格/卡片”后，数据总量与分页一致。
2. 卡片视图操作按钮与表格视图行为一致。
3. 筛选条件变化后，两种视图都能显示同一结果集。
4. 分页（上下控件）在两种视图下均可用。
