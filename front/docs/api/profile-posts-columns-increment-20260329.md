# 个人主页「我的帖子」列表列调整增量文档（2026-03-29）

## 1. 变更目标

个人主页“我的帖子”仅展示已发布帖子后，去掉“状态”列，改为互动数据展示，便于用户快速查看帖子表现。

页面：`/admin/profile` -> `我的帖子`

## 2. 前端已完成

列表列调整为：

1. `ID`
2. `标题`
3. `点赞`
4. `收藏`
5. `评论`
6. `更新时间`

说明：
- 前端对 `likeCount/favoriteCount/commentCount` 做了空值兜底（默认 `0`）。

## 3. 后端接口增量要求

接口：`GET /api/v1/profile/me/posts`

每条帖子建议补充字段：

- `likeCount`（number）
- `favoriteCount`（number）
- `commentCount`（number）

其余维持已有字段（`id,title,updatedAt,...`）。

## 4. 验收项

1. 个人主页“我的帖子”不再显示状态列。
2. 可看到点赞/收藏/评论三项计数。
3. 三项计数字段缺失时前端显示 `0`，页面不报错。
